## Run Locally

Clone the project

```bash
  https://github.com/ivan-escribano/gif-life.git
```

Go to the project directory, if you are not
```bash
  cd gif-life
```

Install dependencies in frontend and backend 

```bash
  npm install
```

Start the server in frontend

```bash
  npm run dev
```

Start the server in backend
```bash
  npm run server
```
